# ТЕХНИЧЕСКОЕ ЗАДАНИЕ  
## **Мини-проект: Консольное приложение "Крестики-нолики" с алгоритмом Минимакс**

> **Цель проекта:**  
> Создать **консольное приложение**, которое принимает текущее состояние игрового поля 3×3 и **находит оптимальный ход** для текущего игрока, используя **алгоритм Минимакс**.  
> Программа **выводит обновлённое поле с уже вставленным ходом** и **начисляет очки** в зависимости от результата.

---

## 1. Описание задачи

Программа:
- Принимает **ввод** — текущее состояние поля (3×3).
- **Определяет**, кто должен ходить следующим (`X` или `O`).
- Находит **оптимальный ход** для этого игрока с помощью **алгоритма Минимакс**.
- **Выводит новое поле**, в котором сделан ход.
- **Выводит начисленные очки**:
  - `+100` — если **ход не завершает игру**,
  - `+100` — если **ход завершает игру и выигрывает X**,
  - `-100` — если **ход завершает игру и выигрывает O**,
  - `0` — если **ход завершает игру и это ничья**.

---

## 2. Входные данные

Программа принимает:
- Строку из 9 символов: `'X.O.X..O.'`
- Или 3 строки по 3 символа:

    X.O

    .X.

    .O.
- Символы: `'X'`, `'O'`, `'.'` (пустая клетка).

---

## 3. Логика

- **Определение игрока**:
  - Если `X` и `O` равны → ходит `X`.
  - Если `X > O` → ходит `O`.
- **Алгоритм Минимакс**:
  - Рекурсивно перебирает все возможные ходы до конца игры.
  - Оценка:
    - +10 — выигрыш `X`,
    - -10 — выигрыш `O`,
    - 0 — ничья.
  - Находит **оптимальный ход** для текущего игрока.
  - После нахождения хода:
    - **Вставляет** символ в поле,
    - **Выводит** обновлённое поле,
    - **Определяет**, завершает ли ход игру:
      - Если **да**, выводит `+100`, `-100` или `0` в зависимости от результата,
      - Если **нет**, выводит `+100`.

---

## 4. Выходные данные

- **Новое игровое поле** (в формате 3 строки по 3 символа).
- Сообщение:
  - `+100 очков` — если игра продолжается или выигрывает X,
  - `-100 очков` — если выигрывает O,
  - `0 очков` — если ничья.
- Если игра уже завершена:
  - Вывод: `Игра завершена. Результат: X.` (или `O`, или `Ничья`)
  - Затем:
    - `+100 очков` — если результат `X`,
    - `-100 очков` — если результат `O`,
    - `0 очков` — если результат `Ничья`.

---

## 5. Предупреждения и всевозможные дисклеймеры
 - Сразу беру во внимание то, что X - начинает игру именно с самого начала, то есть, если бы у нас было пустое поле, то мы бы получили X в верхней левой клетке. 
- Также гарантированно пользователь вводит правильные данные, без ошибок и так далее. 
- Запуск программу производился из Visual Studio Code.
- Для успешного запуска установите python с официального сайта (вбейте в поисковике `download the newest version of python`)
- Проверьте в терминале / командной строке работоспособность `python --version`
- Скачайте и откройте репозиторий в Visual Studio Code
- Запустите программу в файле `main.py`. Можно сделать это как нажав кнопку запуска, так и через командную строку `python main.py`
